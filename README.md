## 1、项目介绍

基于springboot的在线管理系统2拥有三种角色

管理员：用户管理、试卷分类管理、试题管理、添加试题、试题分类管理、考试管理、发布试卷、设置参加考试的学生、设置参加判卷的老师等

教师：判卷

学生：搜索试卷、答题


## 2、项目技术

后端框架：springboot

前端技术：Bootstrap、jsp、css、JavaScript、JQuery

## 3、开发环境

- JAVA版本：JDK1.8
- IDE类型：IDEA、Eclipse都可运行
- tomcat版本：Tomcat 9+
- 数据库类型：MySql（5.x和8.x版本都可） 
- maven版本：无限制
- 硬件环境：Windows 或者 Mac OS
